# 🚗 Road Accident Analysis Dashboard

## 🚀 Project Overview

This is my **third Excel dashboard project**, focused on analyzing **road accident data** to uncover accident patterns, severity levels, vehicle impact, and environmental conditions.

Unlike previous projects, this dashboard uses a **single consolidated dataset** instead of multiple fact and dimension tables, which means **no data modeling was required**.

The dashboard is fully interactive and provides detailed insights across:

* **Accident severity**
* **Vehicle types involved**
* **Road & environmental conditions**
* **Time trends**

Navigation and user interaction are implemented using Excel's **link feature**, enabling smooth movement between sections and external resources.

---

## 🛠️ Tools & Technologies Used

* Microsoft Excel
* Power Query (ETL Process)
* Pivot Tables
* Pivot Charts
* Slicers & Interactive Filters
* Shapes & Custom UI Design
* Link Navigation (worksheets, email, GitHub)

---

## ⚙️ Project Workflow

### 🔹 Data Processing (ETL)

* Imported the road accident dataset into **Power Query**
* Performed data cleaning, transformations, and standardization
* Loaded clean data into Excel for analysis

### 🔹 Data Modeling

* A single dataset was used, so **no relationships or data modeling were required**
* All KPIs and insights were derived directly from the cleaned dataset

### 🔹 Data Analysis

* Created **Pivot Tables** to analyze:

  * Accident counts
  * Severity levels
  * Vehicle-wise accidents
  * Environmental and road conditions
* Applied aggregation logic to surface meaningful trends

### 🔹 Dashboard Design

* Designed a clean and professional UI using **shapes and layout planning**
* Organized visuals into logical sections for better storytelling

### 🔹 Interactivity

* Added slicers and filters for:

  * Year
  * Weather conditions
  * Wind conditions
* Implemented dynamic filtering across all visuals
* Added **navigation links** to:

  * Move between sheets
  * Open GitHub profile
  * Send an email

## 📊 Dashboard Features

### 🔹 Accident Overview

* Total number of accidents
* Deaths
* Serious injuries
* Slight injuries
* Accidents involving cars

### 🔹 Accident Analysis

* Monthly accident trends (Current Year vs Previous Year)
* Accidents by vehicle type
* Accidents by road type
* Accidents by road surface
* Accidents by weather conditions

### 🔹 Environmental & Location Insights

* Urban vs Rural accident distribution
* Accidents by light conditions (Daylight vs Darkness)
* Accidents by wind conditions

### 🔹 KPIs

* Total Accidents
* Total Deaths
* Total Serious Injuries
* Total Slight Injuries
* Accidents by vehicle type (Car, Bike, Bus, etc.)
